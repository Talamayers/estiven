Conceptos de React

Cuadro Comparativo entre React y JSX

React es una biblioteca para construir interfaces de usuario, mientras que JSX es una extensión de sintaxis para escribir elementos de React que se asemejan a XML.

¿Qué son clases en React?

Las clases en React son una forma de definir componentes utilizando la sintaxis de clases de JavaScript. Sin embargo, con la introducción de React Hooks, también es posible crear componentes funcionales.

Principales eventos de React

Algunos eventos fundamentales en React incluyen onClick, onChange, onSubmit, onMouseOver, entre otros, que se utilizan para manejar interacciones con elementos de la interfaz de usuario.
